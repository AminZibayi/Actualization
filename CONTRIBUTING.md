# Contributing

Thanks for your interest in contributing to this project!

## How to contribute

- Fork the repository and create a topic branch for your changes.
- Open a pull request against `main` with a clear description of your change.
- Run tests and linters locally before submitting.

## Local development

- Install dependencies: `pnpm install`
- Run the dev server: `pnpm dev`
- Run tests: `pnpm test`

## Coding guidelines

- Follow the existing TypeScript and React patterns used in the repo.
- Keep changes small and focused; add tests for new behavior.
